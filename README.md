# FARTER Live V3

This is the live-ready version.

It uses:
- Vercel for hosting
- Supabase Database for posts/messages
- Supabase Realtime for live updates
- Supabase Storage for audio files

Important:
- Do not upload a real `.env` file to GitHub.
- Put `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` only in Vercel > Environment Variables.
- The AI verification is simulated for now.
